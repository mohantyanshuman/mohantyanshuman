## Featured

### [LocalPulse](https://github.com/mohantyanshuman/localpulse) &middot; live at [localpulse.dmj.one](https://localpulse.dmj.one)

An AI crisis helpline for small towns. Call it, talk like you would to a neighbor, in five languages, and it answers using live satellite and ground data.

Inside: worldwide multi-sensor satellite fusion across 13 Earth-observation feeds, a self-learning forecast engine, an agentic Gemini voice line with GPS-aware tool use, evacuation-route clearance, and cryptographically signed warning certificates that anyone can verify offline. Single Node process, Cloud Run scale-to-zero, three npm dependencies, 117 automated tests, deployed continuously from GitHub.

Built as my B.Tech capstone at Shoolini University.
