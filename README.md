### Featured project &middot; LocalPulse

<a href="https://localpulse.dmj.one"><img src="https://localpulse.dmj.one/img/ui-satellite-dark.jpg" alt="LocalPulse: Satellite Intelligence panel showing per-hazard satellite fusion, confidence meters and offline cryptographic verification" width="100%"></a>

An AI crisis helpline for small towns. Call it, talk like you would to a neighbor, in five languages, and it answers using live satellite and ground data.

Inside:
- Worldwide multi-sensor satellite fusion across 13 Earth-observation feeds with cross-validation and anti-spoof attenuation.
- A self-learning forecast engine that grades its own past warnings against real outcomes.
- An agentic Gemini voice line: continuous hands-free call, GPS-aware, can file a report or mark you safe on your spoken confirmation.
- Evacuation-route clearance with a GO / CAUTION / NO-GO verdict.
- Cryptographically signed warning certificates that anyone can verify offline in a browser, with no server.

Single Node.js process, Cloud Run scale-to-zero, three npm dependencies, 117 automated tests, deployed continuously from GitHub.

Built as my B.Tech capstone at Shoolini University &middot; [Repo](https://github.com/mohantyanshuman/localpulse) &middot; [Live site](https://localpulse.dmj.one) &middot; [Pitch deck](https://localpulse.dmj.one/pitch)
